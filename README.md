# PlantPathologyClassification
The AI tool for classification of apple trees' diseases based on the photographs of the their leaves. <br><br>
The project has been developed as a part of Kaggle Tournament Plant Pathology 2021 - FGVC8 (after its finish). <br>
Final result the best model achieved to score id 0.822 on full test set, while best solution presented was 0.88.
